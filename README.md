# All Star Proz Barbershop

Official website for All Star Proz Barbershop.

## Project Overview

This project serves as the online presence for All Star Proz Barbershop, providing customers with information about services, pricing, booking appointments, business hours, and contact details.

## Features

- Responsive Design
- Online Appointment Booking
- Services & Pricing
- Barber Profiles
- Photo Gallery
- Customer Testimonials
- Contact Information
- Google Maps Integration
- Mobile Friendly

- CSS3

- Tailwind CSS (if applicable)
